# Sightline Trailer Design

## Positioning

Sightline is a synthetic wind tunnel for physical advertising: a fast, agentic OOH workflow that turns a company URL into creative, inventory, placement scoring, simulated street exposure, and an executable media brief.

## Visual World

Urban outdoor media command center. The video should feel like a roadside billboard, a planning war room, and a transit control board merged together. Use city-map geometry, billboard structures, measurement lines, paper proof marks, route paths, and inspection overlays. Avoid generic neon HUDs, gradient text, and decorative blobs.

## Palette

- Asphalt: `#101114`
- Ink: `#17181b`
- Concrete: `#f2efe6`
- Road Marking Yellow: `#ffd34d`
- Signal Red: `#e4483f`
- Map Green: `#58a06b`
- Route Blue: `#3d7fb8`
- White: `#ffffff`

## Type

- Display: `Arial Black`, `Impact`, system fallback
- Interface: `Inter`, `Arial`, sans-serif
- Mono: `Consolas`, `Courier New`, monospace

## Motion

Sharp lateral wipes, route-line draws, billboard flips, inspection zooms, and metric ticks. Pacing pattern: fast-fast-hold-fast-simulation-punch-lockup.

## Creative Prompting Rubric

Do not ask for “a product trailer with cards.” Ask for a concrete world:

> Create a HyperFrames trailer that feels like a physical OOH pre-flight lab. Use roadside billboard structures, street geometry, route rays, stamps, proof boards, and simulation evidence. Every scene needs foreground, midground, and background. Avoid centered card grids, cyan/purple neon, generic dashboards, gradient text, and repeated equal-size panels. Make transitions feel like camera movement through a city planning machine.

Quality bar:
- Each beat must have a visual metaphor, not only a headline.
- Use no more than one repeated layout pattern.
- Treat screenshots as evidence objects inside the world, not flat pasted images.
- Use strong asymmetry, physical props, and type that occupies real frame space.
